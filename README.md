# 🏡 Helia

Helia é um aplicativo de **aluguel de casas** desenvolvido com **React Native** e **TypeScript**, oferecendo uma experiência fluida, moderna e intuitiva para quem busca encontrar sua próxima moradia de forma rápida e prática.

---

## ✨ Funcionalidades

- 🔍 Busca inteligente de imóveis
- 🏠 Lista de casas disponíveis para aluguel
- 💬 Ícone de chat para suporte e contato com os proprietários
- 🔔 Notificações de novidades
- 📌 Marcar imóveis como favoritos

---

## 📱 Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Expo](https://expo.dev/)
- [Phosphor Icons](https://phosphoricons.com/) para os ícones
- [React Navigation](https://reactnavigation.org/) para navegação entre telas
